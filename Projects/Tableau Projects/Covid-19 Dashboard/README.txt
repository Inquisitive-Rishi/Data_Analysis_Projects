This is project done on covid-19 dataset with interactive dashboard.

Sections of dashboard contains:
	
	1.Covid-19 confirmed cases by country.
	2.Recovered cases by country.
	3.Confirmed cases and recovery cases by Indian states.
	4.Top 5 states with highest deaths.
	5.Forecast of Covid cases (internationally)
	6.Variations in covid cases based on countries.

The dashboard can be viewed via this link : https://public.tableau.com/authoring/RishiRaj/Dashboard1#2
