Titanic Dataset Analysis:

This includes common data exploration techniques such as loading, viewing using (head,tail).

Data is then cleaned off null values and visualized using Heatmap.

Fields having null values > 70% were dropped entirely.
rest fields were replaced dummy values or avg values in case of numeric values.

Analysis also includes Categorical Encoding w.r.t "Sex" column having only two category of either Male or Female.

Univariate analysis was done on the dataset to calculate certain percentages and using boxplot to find outliers.
Bivariate analysis includes comparing survival rates by 'sex' and 'pclass' and basic visualizations.

This analysis also includes basic feature engineering that was done on multiple columns.
